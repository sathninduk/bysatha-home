Question 1:

Create a view to show the title of the movie being shown, theater name, country and city for shows which the theater is fully booked.

CREATE VIEW FullyBookedShows AS
SELECT S.movieTitle, S.theaterName, T.country, T.city
FROM Show S
JOIN Theater T ON S.theaterName = T.theaterName
WHERE S.spectators >= T.capacity;

Question 3:

Create a function which returns the total earning given a movie title.

(movieTitle is the input parameter)

CREATE FUNCTION GetTotalEarning(movieTitle CHAR(25))
RETURNS REAL
AS
BEGIN
    DECLARE totalEarning REAL;

    SELECT @totalEarning = SUM(ticketPrice * spectators)
    FROM Show
    WHERE movieTitle = @movieTitle;

    RETURN @totalEarning;
END;
